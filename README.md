# Vuln Analyzer

**Vuln Analyzer** — це Python-утиліта для сканування вразливостей у Docker-контейнерах, директоріях або репозиторіях за допомогою **Trivy**.  
Інструмент допомагає швидко перевіряти рівень безпеки середовища та формувати звіт про знайдені проблеми.

---

## Основні можливості
- Автоматичне сканування директорій, образів Docker або Git-репозиторіїв  
- Фільтрація результатів за рівнем серйозності (`LOW`, `MEDIUM`, `HIGH`, `CRITICAL`)  
- Експорт результатів у JSON-файл  
- Простий CLI-інтерфейс для інтеграції в CI/CD

---

## Вимоги
- **Python 3.10+**
- **Trivy** (встановлений і доступний у системному PATH)
- Підтримка Linux, macOS або Windows

---

## Встановлення
```bash
git clone https://github.com/mmatiishin/vuln_analyzer.git
cd vuln_analyzer
pip install -r requirements.txt

## Використання

Запуск сканування з вказанням директорії або Docker-образу:

py scanner.py . --min-severity HIGH --output report.json


Аргументи:

subject — шлях до директорії, контейнера або репозиторію

--min-severity — мінімальний рівень вразливостей для звіту

--output — шлях до файлу для збереження результатів
